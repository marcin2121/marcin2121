## 👨‍💻 About Me

I am an **Agentic React Developer** for whom programming is a continuous search for the best and most optimal solutions. I thrive on the bleeding edge of web development, building scalable applications with **Next.js**, **TypeScript**, and **Supabase**. 

My daily workflow is drastically accelerated by **AI-assisted coding** (Claude Code, Antigravity). I don't just write code; I orchestrate it. I have hands-on experience in building highly optimized **PWAs** (Offline-first, Background Sync, Web Push) and integrating LLMs via **Vercel AI SDK**.
